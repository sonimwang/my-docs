# AI Prompt Engineering 基础

## 什么是 Prompt Engineering

Prompt Engineering 是通过设计清晰、结构化的指令，让 AI 模型产出更准确、更有用的回答。

## 我的实践经验

在文档翻译和校对工作中，我使用 Claude Code 完成了以下任务：

### 中英文翻译

```text
将以下 RST 文档翻译为中文，要求：
- 技术名词保留英文（如 "KV cache"、"ZMQ"）
- 代码块和链接不翻译
- 保持 RST 语法完整
```

### 文档校对

```text
审查以下机器翻译的 .po 文件，人工校对并修正以下问题：
- 术语翻译不一致
- 语法错误
- RST 标记损坏
```

### 内容生成

```text
为以下 API 编写中文使用指南，包含：
- 功能概述
- 前置条件
- 步骤说明
- 代码示例
```

## 基本技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **角色设定** | 告诉 AI 它是谁 | "你是一名技术文档工程师" |
| **明确格式** | 指定输出格式 | "用 Markdown 表格列出" |
| **分步拆解** | 复杂任务拆分 | "先做 A，再做 B，最后做 C" |
| **提供范例** | 给一个参考样本 | "参考以下风格：" |
| **限定范围** | 缩小关注范围 | "只检查 RST 语法错误" |
