# 小米路由器 RR4A 产品说明书

## 1. 产品概述

小米路由器 R4A 是一款双频家用无线路由器，支持 2.4GHz 和 5GHz 双频 Wi-Fi，最大无线速率 1167Mbps（2.4GHz 300Mbps + 5GHz 867Mbps）。适用于家庭、小型办公室等场景，最多支持 64 台设备同时连接。

**技术规格**

| 项目 | 参数 |
|------|------|
| 型号 | 小米路由器 R4A |
| Wi-Fi 标准 | IEEE 802.11 a/b/g/n/ac |
| 无线频段 | 2.4GHz / 5GHz 双频 |
| 天线数量 | 4 根外置全向天线 |
| WAN 口 | 1 个 100Mbps |
| LAN 口 | 2 个 100Mbps |
| 电源规格 | 12V / 1A |
| 工作温度 | 0°C – 40°C |

## 2. 开箱与安装

**包装内容**

- 小米路由器 R4A × 1
- 电源适配器 × 1
- 网线 × 1
- 说明书 × 1

**安装要求**

- 宽带入户已开通（光猫或入户网线可用）
- 智能手机一台（用于初始配置）

**接线步骤**

1. 将路由器放置在开阔位置，天线竖直展开
2. 将电源适配器插入路由器电源口，另一端插入插座
3. 将入户网线（或光猫 LAN 口引出的网线）插入路由器蓝色 **WAN 口**
4. 如有有线设备，用网线连接路由器灰色 **LAN 口**

> **注意**：路由器通电后，前面板指示灯为橙色表示等待配置，蓝色表示已连接互联网。

## 3. 连接与配置

**方法一：通过小米路由器 App 配置（推荐）**

1. 手机连接路由器默认 Wi-Fi（SSID 为 `Xiaomi_XXXX`，无需密码）
2. 下载"小米 WiFi"App
3. 打开 App，点击"添加路由器"
4. 设置 Wi-Fi 名称（SSID）和密码
5. 设置管理员密码（可与 Wi-Fi 密码相同）
6. App 提示"配置成功"后，用新密码重新连接 Wi-Fi

**方法二：通过浏览器配置**

1. 手机/电脑连接路由器默认 Wi-Fi
2. 浏览器地址栏输入 `miwifi.com` 或 `192.168.31.1`
3. 点击"开始设置"
4. 按照页面提示设置 Wi-Fi 名称、密码和管理员密码
5. 设置完成后，用新密码重新连接 Wi-Fi

## 4. 常见问题

**Q1：忘记管理员密码怎么办？**

在路由器通电状态下，用针按住路由器底部的 **Reset 孔**约 5 秒，指示灯闪烁后松开。路由器恢复出厂设置，使用默认 Wi-Fi 或无密码连接重新配置。

**Q2：Wi-Fi 信号弱或断连？**

- 检查路由器是否被金属物遮挡或放在墙角
- 尝试切换 Wi-Fi 频段（5GHz 速度快但穿墙弱，2.4GHz 穿墙强）
- 远离微波炉、蓝牙设备等 2.4GHz 干扰源
- 在 App 内查看信道拥挤情况，手动切换信道

**Q3：无法访问互联网？**

排查步骤：
1. 检查光猫是否正常工作（指示灯状态）
2. 检查 WAN 口网线是否松动
3. 打开路由器管理页面，检查 WAN 口连接状态
4. 如显示"未连接"，联系宽带运营商确认入户线路状态
