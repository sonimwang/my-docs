# Git 进阶

## 常用命令

```bash
git log --oneline   # 看历史提交记录
git diff            # 查看当前改了什么（还没 git add 和 commit）
```

## Git 名词解释

| 名词 | 解释 |
|------|------|
| repo | 仓库，一个被 Git 管理的项目文件夹 |
| commit | 提交，"拍照存档"，给当前文件状态拍一张快照 |
| branch | 分支，可随便改，不影响 main |
| merge | 合并分支，把分支上的改动并入 main |
