# Xiaomi Router R4A – Product Manual

## 1. Product Overview

Xiaomi Router R4A is a dual-band home wireless router supporting both 2.4GHz and 5GHz Wi-Fi, with a maximum wireless speed of 1167Mbps (300Mbps on 2.4GHz + 867Mbps on 5GHz). Designed for home and small office use, it supports up to 64 devices simultaneously.

**Technical Specifications**

| Item | Specification |
|------|---------------|
| Model | Xiaomi Router R4A |
| Wi-Fi Standard | IEEE 802.11 a/b/g/n/ac |
| Frequency Bands | 2.4GHz / 5GHz dual-band |
| Antennas | 4 external omnidirectional antennas |
| WAN Port | 1 × 100Mbps |
| LAN Ports | 2 × 100Mbps |
| Power Supply | 12V / 1A |
| Operating Temperature | 0°C – 40°C |

## 2. Unboxing and Installation

**Package Contents**

- Xiaomi Router R4A × 1
- Power adapter × 1
- Ethernet cable × 1
- User manual × 1

**Prerequisites**

- Broadband service is active (optical modem or wall Ethernet port available)
- A smartphone for initial setup

**Cable Connection**

1. Place the router in an open area with the antennas extended vertically
2. Connect the power adapter to the router's power port and plug it into a power outlet
3. Connect the incoming Ethernet cable (from the optical modem LAN port) to the blue **WAN port** on the router
4. For wired devices, use an Ethernet cable to connect to the gray **LAN ports**

> **Note**: After the router powers on, the front LED indicator is orange while waiting for configuration and turns blue once connected to the internet.

## 3. Connection and Setup

**Option A: Using the Xiaomi WiFi App (Recommended)**

1. Connect your phone to the router's default Wi-Fi (SSID: `Xiaomi_XXXX`, no password required)
2. Download the "Xiaomi WiFi" app
3. Open the app and tap "Add Router"
4. Set a Wi-Fi name (SSID) and password
5. Set an administrator password (can be the same as your Wi-Fi password)
6. Once the app shows "Setup Complete", reconnect to Wi-Fi using the new password

**Option B: Using a Web Browser**

1. Connect your phone or computer to the router's default Wi-Fi
2. In a browser, enter `miwifi.com` or `192.168.31.1`
3. Click "Start Setup"
4. Follow the on-screen prompts to set your Wi-Fi name, password, and administrator password
5. Once complete, reconnect to Wi-Fi using the new password

## 4. Troubleshooting

**Q1: Forgot the administrator password?**

With the router powered on, use a pin to press and hold the **Reset** button on the bottom of the router for approximately 5 seconds until the LED indicator blinks, then release. The router will restore to factory settings. Reconnect using the default Wi-Fi (no password) and repeat setup.

**Q2: Weak Wi-Fi signal or frequent disconnections?**

- Check if the router is blocked by metal objects or placed in a corner
- Try switching frequency bands (5GHz offers faster speeds but weaker wall penetration; 2.4GHz has stronger wall penetration)
- Keep the router away from 2.4GHz interference sources such as microwave ovens and Bluetooth devices
- Check channel congestion in the app and manually switch to a less crowded channel

**Q3: No internet access?**

Troubleshooting steps:
1. Check that the optical modem is working properly (indicator status)
2. Check that the WAN port Ethernet cable is firmly connected
3. Open the router management page and verify WAN port connection status
4. If "Disconnected" is displayed, contact your broadband service provider to check the incoming line
