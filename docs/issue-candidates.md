## 候选 3

---

### 打开后怎么看

1. 页面中间是一排 Issue 列表
2. 先看标签，找带"翻译""文档""good first issue"的
3. 点进去，拉到最下面看评论区
4. 评论区空着 = 没人占 → 在评论区发 "我想认领这个任务"
5. 评论区有人说"认领/我来做" → 跳过，找下一个
